# Python Geocoding API Tool

A Python-based utility that interacts with the Geoapify API proxy to convert addresses into geographic coordinates (latitude/longitude) and Plus Codes.

## Features
- Modular code structure using Python functions.
- Secure SSL handling for API requests.
- Error handling for missing data or network issues.
- JSON parsing of nested API responses.

## How to Use
1. Run the script: `python geocoding.py`
2. Enter any city or address (e.g., "Casablanca" or "Eiffel Tower").
3. View the formatted address and coordinates.

## Technologies Used
- Python 3
- `urllib` (Network requests)
- `json` (Data parsing)
